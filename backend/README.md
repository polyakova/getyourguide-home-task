# Backend

The backend app that serves activities functionality.

## Run a Full Build (compile, lint, tests)

```sh
./gradlew build
```

### Run Tests

```sh
./gradlew test
```

### Linting

```sh
./gradlew ktlintformat
```

To check if there are linting issues:
```sh
./gradlew ktlintCheck
```

### Run

```sh
./gradlew bootrun
```
